========================================================================
  03. STRUCTURE PRINCIPLES
========================================================================

  These five rules govern every piece of copy we ship.

  ── 3.1  FRAGMENT OVER SENTENCE ──────────────────────────────────────

    YES:  Gold Star Zard. PSA 9. Clean AF. WOTC grail. No BS.
    NO:   This is a professionally graded Charizard card from the
          vintage era in excellent condition.

    Fragments move faster. Collectors scan, they don't read essays.

  ── 3.2  HARD STOPS ──────────────────────────────────────────────────

    Short. Sharp. Final.
    No weak connectors. No meandering clauses.
    Every period is a mic drop.

  ── 3.3  ALL CAPS FOR IMPACT ─────────────────────────────────────────

    Use deliberately. Sparingly. For PEAK ENERGY moments only.
    If everything screams, nothing does.

    Rule of thumb: one caps phrase per block of copy. Max.

  ── 3.4  BLOCK TYPOGRAPHY THINKING ───────────────────────────────────

    Write like you're designing UI.
    Every line should hit like a bold heading.
    If it doesn't work as a standalone line on a card, rewrite it.

  ── 3.5  CUT RUTHLESSLY ─────────────────────────────────────────────

    If a word doesn't push weight, delete it.
    No padding. No qualifiers.

    Kill list: "just", "really", "very", "actually", "basically",
    "honestly", "simply", "quite", "literally"

    Exception: "actually" is permitted when proving authenticity.
    "No BS — this actually happened."

  11.4  SLAB DESCRIPTION FORMAT
------------------------------------------------------------------------

  When describing a graded card, always include these six elements
  in this order:

  1. Pokemon name
  2. Set name or set number
  3. Grade (PSA 10, BGS 9.5, etc.)
  4. Key identifiers (1st edition, shadowless, holo, reverse, etc.)
  5. Notable features (centering, pop count, condition highlights)
  6. Honest flaw disclosure (No BS)

  Example:
  CHARIZARD — Base Set #4 — PSA 9
  1st Edition. Holo. Centered 55/45.
  Minor whitening top edge. Otherwise clean AF.
  Pop 287 in 9. Comps at $42K. No BS.
