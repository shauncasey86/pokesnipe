====================================================================
                         SCRYDEX API AUTHENTICATION
====================================================================

OVERVIEW
--------------------------------------------------------------------
Authentication is required to access the Scrydex API. Follow the
steps below to set up access and make authenticated requests.

--------------------------------------------------------------------
STEP 1 — CREATE A SCRYDEX ACCOUNT
--------------------------------------------------------------------
Register for an account at:
https://scrydex.com/register

After registering, subscribe to one of the available Scrydex plans.

--------------------------------------------------------------------
STEP 2 — CREATE A TEAM
--------------------------------------------------------------------
A Scrydex Team is a group of users who share access to the same API key.
You can create a team from the Scrydex Account Hub.

IMPORTANT:
• Your Team ID must be included in all API requests.
• Even individual users must create a team.
• Team IDs are unique.

--------------------------------------------------------------------
STEP 3 — GENERATE AN API KEY
--------------------------------------------------------------------
After subscribing to a plan, generate an API key from the dashboard.

Your API key controls:
• API credit limits based on your subscription plan
• Overage billing if credits are exceeded

--------------------------------------------------------------------
STEP 4 — KEEP YOUR API KEY SECURE
--------------------------------------------------------------------
Your API key is sensitive. Always follow these rules:

• Never expose API keys in public repositories (ex: GitHub)
• Never place API keys in client-side code
• If compromised, immediately rotate the key in the Account Hub

--------------------------------------------------------------------
REQUIRED ITEMS FOR API ACCESS
--------------------------------------------------------------------
You MUST have all three of the following:

• Scrydex Team ID
• Scrydex API Key
• Active Scrydex Plan

====================================================================
                    MAKING AUTHENTICATED API REQUESTS
====================================================================

AUTHENTICATION METHOD
--------------------------------------------------------------------
Include the following headers in EVERY API request:

• X-Api-Key
• X-Team-ID

--------------------------------------------------------------------
HTTPS REQUIREMENT
--------------------------------------------------------------------
• All requests MUST be made over HTTPS
• HTTP requests will fail or auto-redirect to HTTPS
• Requests without authentication headers will still work,
  but will be heavily rate limited

--------------------------------------------------------------------
EXAMPLE REQUEST
--------------------------------------------------------------------
curl -X GET "https://api.scrydex.com/pokemon/v1/cards" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Team-ID: YOUR_TEAM_ID"

====================================================================
