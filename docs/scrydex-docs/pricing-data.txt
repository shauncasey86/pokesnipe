OVERVIEW
--------
The Scrydex API provides detailed trading card pricing data including:
- Raw card pricing
- Graded card pricing
- Historical trends
- Multi-currency market pricing

To include pricing data in responses, use:
?include=prices

Example:
GET /pokemon/v1/cards?include=prices

PRICING METRICS

| METRIC         | DESCRIPTION                                  |

+----------------------------------------------------------------------------------+
|                                   RAW PRICING                                    |
+----------------------------------------------------------------------------------+

Raw pricing reflects physical card condition value.

CONDITIONS

+------+----------------------+
| CODE | CONDITION            |
+------+----------------------+
| NM   | Near Mint            |
| LP   | Lightly Played       |
| MP   | Moderately Played    |
| HP   | Heavily Played       |
| DM   | Damaged              |
+------+----------------------+


RAW PRICE FIELDS

+--------------+----------------------------------------------+
| FIELD        | DESCRIPTION                                  |
+--------------+----------------------------------------------+
| condition    | Card condition code                          |
| is_perfect   | Always false for raw cards                   |
| is_signed    | Always false for raw cards                   |
| is_error     | Always false for raw cards                   |
| type         | Always "raw"                                 |
| low          | Lowest known price                           |
| market       | Average market price                         |
| currency     | Price currency                               |
| trends       | Historical trend data                        |
+--------------+----------------------------------------------+

+----------------------------------------------------------------------------------+
|                                  GRADED PRICING                                  |
+----------------------------------------------------------------------------------+

SUPPORTED GRADING COMPANIES

- PSA (Professional Sports Authenticator)
- CGC (Certified Guaranty Company)
- BGS (Beckett Grading Services)
- TAG, SGC, others (when available per card)


GRADED PRICING METRICS

+--------------+----------------------------------------------+
| FIELD        | DESCRIPTION                                  |
+--------------+----------------------------------------------+
| low          | Lowest recorded price                        |
| mid          | Median recorded price                        |
| high         | Highest recorded price                       |
| market       | Average market price                         |
| currency     | Price currency                               |
+--------------+----------------------------------------------+


GRADED-SPECIFIC FIELDS

+--------------+----------------------------------------------+
| FIELD        | DESCRIPTION                                  |
+--------------+----------------------------------------------+
| grade        | Numeric grade (ex: 10, 9.5, 9)               |
| company      | Grading company name                         |
| is_perfect   | Perfect/pristine designation flag            |
| is_signed    | Autograph indicator                          |
| is_error     | Error card indicator                         |
| type         | Always "graded"                              |
+--------------+----------------------------------------------+
